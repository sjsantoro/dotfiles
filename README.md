# Dotfiles

This repository contains my dotfiles.

* alacritty
* fish
* neovim
* sway
* tmux
* waybar
* wofi

## Installation

To install, ensure that `git` and `stow` are installed and run the following commands:

```
git clone https://github.com/sjsantoro/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install.fish
```
